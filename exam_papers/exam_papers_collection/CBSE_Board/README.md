## üìã Examination Pattern (Class 12)

### Theory Papers
- **Duration**: 3 hours
- **Maximum Marks**: 70-80 (varies by subject)
- **Practical**: 20-30 marks (for science subjects)

### Section-wise Structure
Most Class 12 papers follow this pattern:
- **Section A**: MCQs (1 mark each)
- **Section B**: Very Short Answer (2 marks)
- **Section C**: Short Answer (3 marks)
- **Section D**: Long Answer (5 marks)
- **Section E**: Case Study Based (4 marks)

- Write neatly and clearly

- Write to the point
- Include relevant diagrams

## üîç How to Download Papers

### Step-by-Step Process

1. **Visit Official Website**
   - Go to https://cbseacademic.nic.in/PreviousYearPapers.html

2. **Select Parameters**
   - Choose examination year (2020-2024)
   - Select class (X or XII)
   - Choose subject

3. **Download Files**
   - Usually 3 sets per subject (Delhi, All India, Foreign)
   - Download all sets for comprehensive practice
   - Save marking schemes too

4. **Organize Files**
   - Create subject-wise folders
   - Name files properly (Year_Subject_Set.pdf)
   - Keep marking schemes with question papers
